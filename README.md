# creative-house
Project web to share ideas and activities to do in quarantine with HTML,CSS and JavaScript.

**Dependencies**
---
- NodeJS
- NPM or Yarn
- SQLite
- Express
- Nodemoon


**Install**
---
`https://github.com/annaflavia-castro/creative-house.git`

`yarn install`

`yarn start`
